¡Bienvenido a Envía!

Esta es una herramienta de combinación de correos. Haz clic en los encabezados a continuación para aprender cómo usar cada pestaña:

📧 Pestaña Cuenta

Configura los ajustes de tu cuenta de correo (servidor SMTP, puerto, usuario y contraseña). También puedes probar tu conexión y guardar/limpiar tu configuración.

Para cuentas de Google Workspace: deberás crear una contraseña de aplicación: https://myaccount.google.com/apppasswords, esta será la contraseña del usuario SMTP.

📝 Pestaña Plantilla

Escribe el correo que quieres enviar. Puedes usar variables como {{nombre}} que serán reemplazadas con datos de tu fuente de datos. El editor de texto enriquecido permite un formato básico.

📊 Pestaña Datos

Importa tus datos desde un archivo CSV o pégalos directamente en la tabla. La primera fila debe ser la de los encabezados, que se usarán como nombres de variables en tu plantilla. Puedes añadir/eliminar filas y columnas, y exportar tus datos.

📎 Pestaña Adjuntos

Sube los archivos que quieras adjuntar a tus correos. Puedes seleccionar adjuntos que se enviarán a todos los destinatarios, o usar una variable en la columna "Adjuntos" de tus datos para especificar diferentes adjuntos para cada destinatario.

👁️ Pestaña Vista Previa

Mira cómo se verán tus correos antes de enviarlos. Puedes navegar por cada correo, realizar ediciones individuales y excluir correos específicos de la combinación de correspondencia.

📈 Pestaña Resultados

Después de enviar los correos, puedes ver los resultados de la operación en esta pestaña, incluyendo el estado de éxito/fallo para cada correo y mensajes de error detallados.
